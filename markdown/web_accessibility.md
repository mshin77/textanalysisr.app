## Keyboard Support

- Tab / Shift+Tab: Navigate forward / backward
- Enter / Space: Activate button or link
- Alt+S: Text-to-speech toggle
- Esc: Close modals
- Skip to main content link available

Follows WCAG 2.1 Level AA standards with full keyboard navigation.

## Screen Reader

- ARIA landmarks and labels
- Live announcements for dynamic content
- Descriptive button and image labels

## Visual Features

- **Dark Mode**: Toggle via moon/sun icon
- **High Contrast**: 4.5:1+ ratio (WCAG AA)
- **Resizable Text**: Up to 200%
- **Reduced Motion**: Respects system preferences

## Translation & TTS

- Translation: 100+ languages (globe icon)
- Text-to-Speech: Auto language detection (speaker icon)
