Named Entity Recognition

Named Entity Recognition (NER) identifies and classifies named entities in text into predefined categories such as people, organizations, locations, dates, and quantities.

Entity Types

  • PERSON, ORG: People, organizations
  • GPE, LOC: Places, locations
  • DATE, MONEY, PERCENT: Temporal, monetary values

Usage

Filter by entity type. Add custom entities for qualitative coding.
