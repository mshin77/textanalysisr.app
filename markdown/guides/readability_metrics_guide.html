Readability Metrics

Readability metrics quantify text complexity using statistical measures of sentence structure and word characteristics to estimate comprehension difficulty.

Metrics

  • Flesch Reading Ease: 0-100 (higher = easier)
  • Flesch-Kincaid: U.S. grade level
  • Gunning Fog, SMOG: Education years needed
  • ARI, Coleman-Liau: Character-based grade level

Usage

Select metrics and optionally group by variables to compare documents.
